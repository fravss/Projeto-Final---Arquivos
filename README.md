##Sistema de Gerenciamento de Alunos
Este é um programa simples em C para gerenciamento de dados de alunos. Ele oferece as seguintes funcionalidades:

Criar Arquivo: Permite criar um arquivo para armazenar os dados dos alunos.

Incluir Aluno: Permite a inclusão de informações sobre novos alunos, incluindo prontuário, nome, curso, e notas. Calcula automaticamente a média e a situação do aluno.

Consultar Aluno por Curso: Possibilita a consulta de alunos com base no curso.

Consultar Aluno por Situação: Permite verificar a situação dos alunos (aprovados, em exame ou reprovados).

Finalizar: Encerra o programa.
